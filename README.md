# Strange Camera

Take weirder photos.

## Use it in a browser

[Strange.Cam](https://strange.cam)

## Use it in OBS

```bash
open -a "OBS" --args --use-fake-ui-for-media-stream
```

## Use it locally

```bash
git clone git@github.com:rileyjshaw/strange-camera.git
cd strange-camera
npm i
npm run dev
```
